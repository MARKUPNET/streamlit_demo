# streamlit_demo
グラフの表示などのデモ

## 仮想環境
```
py -m venv venv_tool
venv_tool\Scripts\activate
```

## セットアップツールのアップグレード
```
py -m pip install -U setuptools
```

## pipアップグレード
```
py -m pip install --upgrade pip
```

## モジュールインストール
```
py -m pip install -r requirements.txt
```

## 起動
```
streamlit run Home.py
```
